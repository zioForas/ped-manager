Come ottenere le credenziali
  1. Vai su developers.facebook.com → Crea app → tipo Business
  2. Aggiungi prodotto Instagram Graph API
  3. Genera un Page Access Token con permessi: pages_manage_posts, instagram_basic, instagram_content_publish
  4. Trova il Page ID dalla pagina FB → Info della pagina
  5. Trova l'IG Account ID da Graph API Explorer → /me/accounts
  6. Metti tutto nel file .env e riavvia
